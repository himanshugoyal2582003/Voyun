VOYUN

About VOYUN

VOYUN is a lightweight, browser-based music player designed for personal and educational use. Developed by Himanshu Goyal, it showcases modern web development techniques while offering an immersive interface for playing locally hosted audio tracks.

Inspired by sleek user interfaces and powered by custom folder navigation, playback features, and dynamic song loading — VOYUN offers a practical demo for anyone learning frontend architecture and media interactions.

It supports features like folder-based browsing, track previewing, scroll animations, developer pages, legal disclaimers, and even upload capabilities in future iterations.

This project is non-commercial and built with love to demonstrate technical creativity and design thinking in a structured web environment.
